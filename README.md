## About

A personal portfolio website styled as an interactive terminal interface. Type commands to navigate and discover content.

## Commands

| Command       | Description                |
| ------------- | -------------------------- |
| `help`        | Show available commands    |
| `whoisridze`  | About me                   |
| `discography` | Music projects             |
| `social`      | Social media links         |
| `contactme`   | Contact information        |
| `banner`      | Display the welcome banner |
| `history`     | View command history       |
| `clear`       | Clear the terminal         |
| `secret`      | ???                        |

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Vercel** - Deployment

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
